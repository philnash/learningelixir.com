Learning Elixir

Elixir is a dynamic, functional language designed for building scalable and maintainable applications. It is built on the Erlang virtual machine and inspired by Ruby to make a fast, concurrent, dynamic, functional and fault tolerant language that is a pleasure to write.

This site is a resource for articles and tutorials for learning Elixir as well as a place to write about experiences with the language.

Getting started with Elixir

Installing Elixir

  • Mac OS X

    $ brew install elixir

Hello World!

Open up Interactive Elixir:

$ iex

and type:

iex> IO.puts "Hello World!"
